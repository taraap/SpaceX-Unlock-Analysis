# SpaceX 解禁这事，我捋了一下

> 个人学习笔记，不构成投资建议 | 区块链技术应用专业


## 发生了什么

SpaceX 8月6号要解禁一大批股票，大概 9.115 亿股，市值算下来 1160 亿美元。股价从6月16号高点已经跌了 37%，市值没了 4250 多亿。

盘前看涨了一点，涨了 3.8%，总算止住了七连跌。


## 解禁时间线

把时间线拉出来是这样的：

| 时间 | 怎么回事 | 流通股总量 |
|---|---|---|
| 现在 | 还在锁着 | 约 6.39 亿股 |
| **8月6日** | **第一批解禁，9 亿多股** | 约 15.5 亿股 |
| 12月初 | 差不多全放出来了 | **53.3 亿股** |

还有个条件触发机制——如果 SpaceX 业绩公布前十天里，有五天股价站上 175.5 美元，就额外再解锁 4.5 亿股。现在股价 125 美元，要涨 46% 才能摸到，基本不太可能。


## 谁在卖？谁没卖？

- **马斯克**：持股 60%，锁到 2027 年，短期不会动
- **早期私募投资者**：成本极低，怎么卖都是赚，这帮人是最有动力抛的
- **空头**：30% 的可交易股份被做空了，账面浮盈 70 亿美元

所以真正要盯着的是早期投资者那批人，马斯克反而不用太担心。


## 我的判断

利空确实不少——解禁规模大、早期投资者要套现、空头已经进场、星舰又出了故障、整个市场资金也在从 AI 概念撤出来。

但也不是完全没有缓冲：解禁是分批放的，不是一天砸完；马斯克的仓位锁着；股价已经先跌了 37%，悲观预期其实消化了不少。


## 跟 Web3 有什么关系

我读下来觉得挺有意思的一点是，SpaceX 这个"分阶段解锁"的设计，本质上跟 Web3 项目的 **Cliff + Vesting（锁仓+线性释放）** 是一回事。区别在于：

| | 传统IPO | Web3 |
|---|---|---|
| 解锁规则 | 写在文件里，人工执行 | **智能合约自动跑，链上可查** |
| 透明度 | 靠SEC披露，有滞后 | **实时可验证** |
| 执行风险 | 存在人为变数 | 代码即法律 |

传统金融在学 Web3 的锁仓设计，但透明度和自动化程度还是差一截。如果以后 SpaceX 这类资产能代币化上链，解禁预期带来的恐慌或许能通过链上仪表盘实时可视化，市场反应可能会理性很多。
